Bienvenido al Explorador Cósmico

Aquí tienes unos consejos para empezar:

  • Haz clic en la pantalla para activar el modo de vuelo (aparecerá una mirilla).
  • Usa el ratón para mirar a tu alrededor.
  • Usa W, A, S, D para moverte, Espacio para subir y Shift para bajar.
  • Apunta con la mirilla a un planeta o constelación para ver su información.
  • Presiona Esc para liberar el ratón y poder usar los menús.
